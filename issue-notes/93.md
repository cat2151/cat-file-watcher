## 技術的詳細 / Technical Details

TOMLの仕様では、以下の2つの形式は等価です:

**インライン配列表記（1行）**:
```toml
files = [{path = "file.txt", command = "echo test"}]
```

**配列テーブル表記（複数行）**:
```toml
[[files]]
path = "file.txt"
command = "echo test"
```

両方とも、`files`キーに対して配列（リスト）の値を生成します。現在の`config_loader.py`は`files`セクションが配列であることを要求しているため、どちらの形式も正しく動作します。
